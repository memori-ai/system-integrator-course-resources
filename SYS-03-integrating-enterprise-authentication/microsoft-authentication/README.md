# Microsoft Authentication

> Integrating Microsoft Azure AD / Entra ID authentication with AIsuru

## Overview

This module covers how to integrate Microsoft authentication (Azure AD / Entra ID) with AIsuru for enterprise Single Sign-On (SSO) scenarios.

## Topics

- Azure AD / Entra ID configuration
- OAuth 2.0 flow with Microsoft identity platform
- Token validation and exchange
- Enterprise SSO integration patterns

## Prerequisites

- Microsoft Azure account
- Basic understanding of OAuth 2.0
- Access to Azure AD / Entra ID admin portal
